# Next Bridge Ventures — Marketing Website UI Kit

Interactive click-through recreation of the NBV marketing site, built from `src/app/page.tsx`, `about/page.tsx`, `faq/page.tsx`, `header.tsx`, `footer.tsx` in the `NextBridgeVentures` repo.

Open `index.html`. Navigate via the header: Home, About Us, FAQ are fully recreated. Programs / Global Entrepreneurs / Canadian Founders render as labeled stubs — their real content (program cards, eligibility criteria, sub-routes) exists in the source repo but wasn't duplicated here; see readme.md caveats.

Files: `Shell.jsx` (Header + Footer), `HomePage.jsx`, `AboutPage.jsx`, `FaqPage.jsx` (5 of the real 18 FAQ items, abbreviated), `StubPage.jsx`.
